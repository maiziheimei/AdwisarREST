# AdwisarREST

This is an example project which is testing the adwisar middrv service.  It simulates a client side to send the server with its machine, machineSchema, machineData. 

## How to run this example
 1. compile it 
   ```
   $ ./compile.sh
   ```
 2. run server
 ```
   $ ./server.sh
   ```
 3. run client
 ```
   $ ./client.sh
   ```
